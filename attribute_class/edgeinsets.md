# EdgeInsets \(padding, magin\)

### 使用方法

```dart
// 全部
padding: EdgeInsets.all(10),

// 方向
padding: EdgeInsets.symmetric(horizontal: 0),

// 單邊
padding: EdgeInsets.only(right: 5.0),
```

### **可使用該屬性的元件**

* Container
* Padding
