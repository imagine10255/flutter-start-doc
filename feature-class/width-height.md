# MediaQuery

### 使用方法

```dart
// 取得螢幕寬高
MediaQuery.of(context).size.width
MediaQuery.of(context).size.height

// 取得螢幕方向
MediaQuery.of(context).orientation

// 確認使用者是否有改變字體大小
MediaQuery.of(context).textScaleFactor

// 取得裝置螢幕的像素比
MediaQuery.of(context).devicePixelRatio
```
