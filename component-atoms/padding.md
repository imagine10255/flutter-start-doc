# Padding

### 使用方法

```text
Padding(
  padding: EdgeInsets.all(8),
  child: Container(),
)
```

### 屬性

| Class | Name | Desc | Default |
| :--- | :--- | :--- | :--- |
| \*[EdgeInsets](../attribute-class/edgeinsets.md) | padding | 內間距 |  |
| Widget | child | 內容 |  |
